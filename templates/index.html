Project Details

This is the Smart Blind Stick Project, created by Jinay Shah and Maahi Patel. The stick is designed to assist visually impaired individuals by providing real-time obstacle detection and sending SOS alerts during emergencies.

The project leverages technologies like ESP32, HTTP protocol, and ultrasonic sensors. It aims to ensure safety and independence for users by connecting seamlessly to a dashboard for live updates.
